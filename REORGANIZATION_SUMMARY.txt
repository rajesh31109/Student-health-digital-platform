COMMANDS - HOW TO RUN
═════════════════════════════════════════════════════════════════════════════════

Frontend Development:
   npm run dev
   → Starts at http://localhost:8080

Backend Development:
   npm run backend:dev
   → Starts at http://localhost:3001

Frontend Build:
   npm run build
   → Compiles to frontend/dist/

Backend Build:
   npm run backend:build
   → Compiles to backend/dist/

Run Both (in separate terminals):
   Terminal 1: npm run dev
   Terminal 2: npm run backend:dev
