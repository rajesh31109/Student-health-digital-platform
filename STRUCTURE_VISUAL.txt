🚀 COMMANDS REFERENCE

Development:
  npm run dev              # Start Frontend    (http://localhost:8080)
  npm run backend:dev      # Start Backend     (http://localhost:3001)

Production Build:
  npm run build            # Build Frontend    (→ frontend/dist/)
  npm run backend:build    # Build Backend     (→ backend/dist/)

Testing & Linting:
  npm test                 # Run Frontend Tests
  npm test:watch           # Watch Mode Tests
  npm run lint             # ESLint Check

Preview:
  npm run preview          # Preview Build
